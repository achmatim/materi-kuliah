Belajar Pemrograman Web: HTML

Apa itu HTML?

HTML merupakan singkatan dari Hyper Text Markup Language. Ibaratnya sebuah rumah, HTML adalah struktur bangunan utama yang membentuk rumah tersebut. Tanpa struktur bangunan, bisa jadi tidak bisa disebut sebagai sebuah rumah. Demikian juga aplikasi web, tidak akan terbentuk tanpa keberadaan HTML.

Dalam HTML, ada 3 istilah yang harus kita pahami:

  1. Tag, yang mewakili setiap komponen di halaman web
  2. Atribut, bagian dari Tag yang mengatur bagaimana komponen ditampilkan
  3. Value, merupakan nilai dari setiap atribut
