HTML

Apa itu HTML?

HTML merupakan singkatan dari Hyper Text Markup Language.

HTML merupakan komponen penting dari sebuah halaman web. HTML merupakan struktur dasar yang membangun halaman web.

Jika Anda ingin menjadi seorang web developer, maka harus belajar HTML.

Istilah di HTML

Setiap komponen HTML terdiri dari 3 hal:

  • Tag
  • Atribut
  • Value

Contoh Representasi Warna di HTML

NO Warna Teks Hexadecimal
1 HITAM black #000000
2 MERAH red #ff0000
3 Hijau green #00ff00
4 Biru blue #0000ff
